# gitbook

### 1. 安装
```bash
# 全局安装gitbook-cli
npm install gitbook-cli -g

# 查看版本
gitbook -V

# 输出
CLI version: 2.3.2
GitBook version: 3.2.3
```

### 2. 初始化项目
```bash
# 初始化
gitbook init

# 输出
warn: no summary file in this book
info: create README.md
info: create SUMMARY.md
info: initialization is finished
```

### 3. 在浏览器中预览
```bash
# 在浏览器中预览
gitbook serve

# 输出
Starting server ...
Serving book on http://localhost:4000

```
